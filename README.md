# Olympics Performance Analytics: Trends and Medalists 🏅

A comprehensive, data-driven web application built with **Streamlit** that explores over 120 years of Olympic history. This project analyzes athlete performance, country-wise achievements, and overall medal trends using interactive visualizations.

🚀 **[Live App Deployment Link]([https://olympics-performance-analytics-trends-and-medalists-6lqtjdw5ku.streamlit.app/])**

---

## 📊 Project Overview
This application provides four distinct levels of analysis:
1. **Medal Tally:** View the overall performance of countries across different years and seasons (Summer/Winter).
2. **Overall Analysis:** High-level statistics on editions, host cities, participating nations, and athlete growth over time.
3. **Country-wise Analysis:** Deep dive into a specific country's performance, including their best years and most successful sports.
4. **Athlete-wise Analysis:** Detailed study of athlete demographics, including age distribution among medalists and Height vs. Weight correlations.

## 🛠️ Tech Stack
* **Language:** Python
* **Web Framework:** Streamlit
* **Data Manipulation:** Pandas, NumPy
* **Visualization:** Plotly, Seaborn, Matplotlib

## 📂 Dataset Information
The analysis is based on the "120 years of Olympic history" dataset.
* **Primary Dataset:** [Download from Kaggle](https://www.kaggle.com/datasets/heesoo37/120-years-of-olympic-history-athletes-and-results)


## ⚙️ Installation & Local Setup
To run this project locally, follow these steps:

1. Clone the repository:

git clone https://github.com/aasthasheth180-ops/Olympics-Performance-Analytics-Trends-and-Medalists.git

2. Navigate to the project folder:


cd Olympics-Performance-Analytics-Trends-and-Medalists

3. Install the required dependencies:


pip install -r requirements.txt

4. Run the Streamlit application:

streamlit run app.py
